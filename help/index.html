How to publish your Maker site

To publish your Maker site, create an account on this: GitHub Then sign up for Vercel.

GitHub + Vercel:

Go here, type in a name for your website, and select public or private. Then create it.
Then, tap "Create new file" and on the right of your repo name at the top, type index.html.
In the big text box below that, paste in what you found in the text box at the bottom of the Maker homepage.
Then tap Commit New File.
Now, go to Vercel and tap New Project. Then under Import Git Repository, tap GitHub. Allow Vercel access, then choose the repository you created.
Tap "Select" under your username, then tap continue.
Don't select a framework, or build command, just tap continue. But make sure to make a project name.
Then, wait a little bit and you should see "Congratulations! Your site has been deployed." or similar. Tap on your site's picture to go to it, and that's it!
